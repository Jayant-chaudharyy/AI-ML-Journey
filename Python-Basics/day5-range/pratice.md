# Python: range() Practice Questions

### 1. The Basics
Generate a sequence of numbers from `0` to `9` using `range()` and convert it into a `list` to print it.

### 2. Defining a Start
Generate a sequence of numbers starting from `10` and ending at `20` (inclusive). Print this as a `list`. 
*(Hint: Remember that the 'stop' parameter is exclusive, so you'll need to go one higher than the number you want to reach.)*

### 3. The Step Parameter
Generate a sequence of only the **odd numbers** between `1` and `20` using the `step` parameter. Print them as a `list`.

### 4. Reverse Counting
Generate a sequence that counts backward from `10` down to `1` (including both `10` and `1`). Print it as a `list`.

### 5. Loop Logic
Use a `for` loop with `range()` to print the phrase `"AI is powerful"` exactly `5` times.
