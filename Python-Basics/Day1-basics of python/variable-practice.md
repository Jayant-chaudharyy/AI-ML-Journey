# Python Day 1: Variables Practice

### 1. Basic String Assignment
Create a variable named `developer_name` and assign your first name to it.

### 2. Basic Integer Assignment
Create a variable named `odometer_reading` and assign the integer `47399` to it.

### 3. Basic Float Assignment
Create a variable named `workout_duration_minutes` and assign the float `45.5` to it.

### 4. Basic Boolean Assignment
Create a variable named `is_jiofiber_connected` and assign the boolean value representing "true" or active.

### 5. Tech Stack Variables
Create three separate variables for a tech stack: `frontend`, `language`, and `database`. Assign them the strings `"Next.js"`, `"TypeScript"`, and `"Firebase"` respectively.

### 6. Snake_case Naming
Create a variable to store the name of a specific AI model (e.g., `"gemini-2.5-flash-lite"`). Ensure you use proper `snake_case` format for the variable name.

### 7. Reassigning a Variable
Create a variable named `slice_account_balance` and set it to `500.0`. On the very next line, update that exact same variable to be `350.50` (simulating a card payment).

### 8. Variable Output
Create a variable called `motorcycle_brand` and assign it a string value of your choice. Then, write the exact Python command you would use to display this variable's value on the screen.

### 9. Storing a Location
Create a variable named `scenic_destination` and assign it a string describing a mountainous river area. 

### 10. Variable Inventory
Create an integer variable for `number_of_punching_bags` and a string variable for `gym_name`. Assign appropriate values to both.
