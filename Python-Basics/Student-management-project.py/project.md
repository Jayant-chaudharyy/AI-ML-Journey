📝 Project Requirements
Your Python script must successfully complete these five tasks using only Variables and Data Types:

Task 1: The Input & Dictionary

Prompt the user to enter a student's Name, Age, and Email using input().

Store these three pieces of data inside a Dictionary called student_profile. Ensure the Age is type-casted to an Integer.

Task 2: The List & Operations

Prompt the user to enter their scores for three subjects out of 100 (e.g., Python, SQL, Statistics).

Store these three scores in a List.

Calculate the Total Marks and the Percentage (Float) using basic math operations on the List items.

Add the Total Marks and Percentage as new key-value pairs into the student_profile Dictionary.

Task 3: The Tuple

Create a Tuple called university_info that contains the University Name ("Bennett University") and the Program ("MCA AI/ML").

Why a Tuple? Because this information is fixed and should not be modified during the script execution.

Task 4: The Set & Intersection

Define a predefined Set called required_ai_skills containing: {"Python", "SQL", "Machine Learning"}.

Prompt the user to enter three skills they currently know, and store them in a new Set called student_skills.

Use a Set operation (Intersection) to find out which required AI skills the student already possesses, and store the result in a variable called matched_skills.

Task 5: The Formatted Output

Use an f-string (formatted string) to print a clean, professional "Student Report Card" to the console that displays all the data gathered from the Dictionary, Tuple, and Set operations.
